Can I use external programs with the driver?

You cannot use an external application with the driver directly. However if you use KisMACs dump feature, you will be able to create pcap file, which can be read by a lot of UNIX style programs. One of the most useful pcap-enabled applications is ethereal, a packet analysis tool. It is available as a fink package (fink.sf.net). Install fink and type in Terminal: "fink install ethereal". Then go into X11 and start it with "sudo ethereal".

Hint: If you want to capture packets of a specific network, you should disable channel hopping - with channel hopping enabled, only packets from the curent channel will be captured, and you will miss data when the card goes to another channel.
