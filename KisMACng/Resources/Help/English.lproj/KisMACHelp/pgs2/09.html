How many weak packets do I need to crack WEP?

Weak Scheduling Attack:

This is a tough question, based on probability: each packet can reveal one certain byte of the key with a probability Currently you will need about a million data packets to be sure that the attack works!

Bruteforce Attack:

The Bruteforce attack does not require any weak keys, however you will need a lot more CPU time to break a key using a bruteforce attack.
